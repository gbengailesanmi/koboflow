# Koboflow Backend

Express.js API server. Handles authentication, transactions, budgets, and integrates with Mono for bank data.

## Stack

- Express.js
- TypeScript
- MongoDB
- Mono API

## Getting Started

```bash
yarn
yarn dev
```

Server runs on [http://localhost:3001](http://localhost:3001)
